# Login API with JWT – Rise Edu Backend

This is a basic backend login API built with Node.js, Express, and MongoDB.  
JWT (JSON Web Tokens) is used for secure authentication.

## 📁 Features

- User registration
- User login
- JWT-based authentication

## 🔧 Technologies Used

- Node.js
- Express
- MongoDB
- Mongoose
- JSON Web Token (JWT)
